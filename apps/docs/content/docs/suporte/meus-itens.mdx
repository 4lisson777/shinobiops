# Meus Itens

A página **Meus Itens** exibe todos os tickets e bugs que você criou, independente do status atual.

## Acessando

No menu lateral, clique em **Meus Itens**.

## O que você vê

- Todos os tickets e bugs criados por você
- Status atual de cada item
- Data de criação e última atualização
- Responsável atribuído (se houver)

## Filtrando

Você pode filtrar por **Tipo** (Ticket / Bug) e **Status** para encontrar itens específicos.

## Reabrindo um item concluído

Se um problema voltou a ocorrer após ser marcado como concluído:

1. Abra o item na lista.
2. Na página de detalhes, clique em **Reabrir**.
3. Adicione um comentário explicando por que está reabrindo.

O item volta para o status **Aberto** e reaparece na fila.
