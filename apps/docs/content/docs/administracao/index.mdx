# Administração

A área de administração é exclusiva para **Tech Leads**. Aqui você configura a equipe, cria convites, ajusta os checkpoints de status e define as regras de notificação.

## Acesso

No menu lateral, clique em **Administração** para expandir as opções:

- **Equipe** — gerenciar membros e convidar novos
- **Checkpoints** — configurar o intervalo de atualização de status
- **Notificações** — definir quem recebe quais alertas
