# Desenvolvimento

A área de desenvolvimento é o espaço de trabalho da equipe técnica. Ela exibe em tempo real o status de cada membro, tickets em andamento e permite coordenação direta entre devs.

## Perfis com acesso

| Perfil | O que pode fazer |
|--------|-----------------|
| Desenvolvedor | Ver o painel, atualizar status, pedir ajuda, pegar tickets |
| QA | Ver o painel, atribuir tickets a devs |
| Tech Lead | Acesso completo |

## Páginas desta seção

- [Painel](/docs/desenvolvimento/painel) — visão geral da equipe em tempo real
- [Atribuição de Tickets](/docs/desenvolvimento/atribuicao) — como pegar ou atribuir tickets
- [Atualização de Status](/docs/desenvolvimento/status) — como informar no que você está trabalhando
- [Pedido de Ajuda](/docs/desenvolvimento/pedido-de-ajuda) — como solicitar e oferecer ajuda
