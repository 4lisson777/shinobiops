# Primeiros Passos

Esta seção cobre tudo que você precisa para começar a usar o VectorOps: como entrar no sistema, criar sua conta e navegar pela interface.

## Pré-requisitos

- Um navegador moderno (Chrome, Firefox, Edge ou Safari)
- Acesso à rede interna da Inovar Sistemas
- Credenciais de acesso ou um link de convite

## Páginas desta seção

- [Login](/docs/primeiros-passos/login) — como entrar na sua conta
- [Cadastro](/docs/primeiros-passos/cadastro) — como criar uma conta com convite
- [Perfil](/docs/primeiros-passos/perfil) — como atualizar seus dados pessoais
