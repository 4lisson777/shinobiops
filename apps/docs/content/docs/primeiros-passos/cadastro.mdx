# Cadastro

O acesso ao VectorOps é por convite. Um **Tech Lead** da sua organização gera um link único que você usa para criar sua conta.

## Cadastro via convite

1. Abra o link de convite enviado pelo seu Tech Lead. Ele tem o formato:
   ```
   https://vectorops.inovar.com/register?invite=XXXXXXXX
   ```
2. A aba **Entrar via convite** será selecionada automaticamente.
3. O sistema exibe as informações do convite: nome da organização e cargo que você receberá.
4. Preencha seus dados:
   - **Nome completo**
   - **Email** (pode estar pré-preenchido e bloqueado se o convite for restrito a um email específico)
   - **Senha** (mínimo de 8 caracteres)
5. Clique em **Criar conta**.

Após o cadastro, você é redirecionado automaticamente para a área do seu perfil.

<Callout type="warn">
  Cada link de convite pode ser usado **uma única vez**. Se o link expirar ou já tiver sido usado, solicite um novo ao seu Tech Lead.
</Callout>

## Criar uma organização

Se você é o primeiro usuário de uma nova organização:

1. Acesse `/register` diretamente.
2. Clique na aba **Criar organização**.
3. Preencha o nome da organização e seus dados pessoais.
4. Você será cadastrado automaticamente como **Tech Lead** da nova organização.

## Validade do convite

Os convites têm um prazo de validade definido pelo Tech Lead (24 h, 48 h, 72 h ou 7 dias). Após esse prazo, o link deixa de funcionar.
